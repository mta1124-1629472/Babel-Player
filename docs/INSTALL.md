# Install and Run (Developer)

1. Install Visual Studio 2022/2023 with .NET Desktop and Windows App SDK workloads.
2. Open `PlayerApp.sln`.
3. Restore NuGet packages.
4. Build and run `PlayerApp.UI`.
5. The app starts with a minimal UI. Open a local video file using the MediaPlayerElement transport controls.
6. Models: use `scripts/download_models.ps1` to manage model downloads. Update the script with real download URLs and checksums.
